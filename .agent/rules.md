# Agent Rules

1. **Context First**: Always review `task.md`, `plan.md`, and `PROGRESS.md` at the start of a task.
2. **Update Documentation**: Keep `task.md` and `PROGRESS.md` specific and up-to-date.
3. **Coding Standards**:
   - Usage of `PropertySheet` and `useLocation` naming conventions.
   - Explicit typing for complex objects (e.g. Zod schemas).
   - Use `npx expo` for running commands.
