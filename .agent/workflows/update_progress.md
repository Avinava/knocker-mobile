---
description: How to read and update project progress
---

1. **Read Status**:
   - Read `task.md` to see active items.
   - Read `PROGRESS.md` to see phase completion status.

2. **Update Status**:
   - If a task is finished, mark it `[x]` in `task.md`.
   - If a phase is complete, update the status table in `PROGRESS.md`.
   - If new tasks are discovered, add them to `task.md`.

3. **Verify**:
   - Ensure `plan.md` matches the actual implementation.
   - Run `tsc` or tests if applicable before marking "Complete".
